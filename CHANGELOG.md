# Changelog

## 0.1.0 — 2026-06

Initial open-source release of Personal Finances Express API (paired with personal-finances).

- `/api/data/*` entity routers (transactions, categories, recurring purchases, loans, …)
- `/api/ai/*` workers (slug assign, category assign, recurring detect)
- Supabase SQL bundle under `docs/supabase/`
- OSS docs: quickstart, database setup, wire contract, CI
